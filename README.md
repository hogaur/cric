## Cric4CWC23

* Aims to develop a cricket trading api. 
* Predict playing 11 for CWC 2023
* API: `teamIndia.playing11(match)`
* Opponent, Venue, Conditions based algo.

## Execute
- `go run main.go`

## Steps
### Roles 
- Get batter list
- Get allrounder list
- Get bowler list
- Get wicketkeeper list
- Get captain list
### Stats

- #matches
- #win percentage
- #wickets
- #stumps
- #bowled
- #30/50/70/100
